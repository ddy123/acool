第二章 温柔的微笑

+A -A
时间：2016-09-29 21:39 字数：2461

    等我睁开眼睛，窗口已经洒满柔和的月光。

    这一觉睡得好舒服……我懒洋洋地伸长双手，在床上滚了两圈，突然发现手机在发光。

    咦，谁在给我打电话？

    我疑惑地点开手机，才发现何诺生居然给我打了十多个电话！一时间没想起录取通知书，我赶紧给他回了电话：“喂喂喂，何诺生吗？对不起我刚才睡着了才没回你电话你别生气啊！！！”

    以前试过一次，因为忘了带手机出去，然后回到家差点被何诺生k死。虽然平时看他挺好欺负，但是一到电话这里他可不管三七二十一，不接就骂得要死，不管过了多久都记仇的……

    “娜娜，爸爸把事情都告诉我了。”没想到何诺生没有生气，而是把声音放轻柔跟我说：“你还想去比格利亚大学念书吗？”

    “唉？！！”我吓得一下子就从床上掉到地上，“……呜呼呼，好痛！”我龇牙咧嘴地摸着自己的屁股，用手撑起身体坐回床边。

    “娜娜，没事吧？”何诺生担心地问我，似乎已经猜到我这边的情况了。

    “没事啦～我何允娜可是宇宙无敌美少女耶！”我轻轻一笑，继续回答他之前的问题：“放心吧，哥哥。我已经想明白了，既然不能进比格利亚大学，我还可以拿自己的入学资料去别的大学嘛！”

    虽然还是有点难过，不过现在的我已经想通了。如果我的梦想要建立在家里人的痛苦上，那又有什么意义呢？还不如换个梦想，哈哈～

    “真的？”一瞬间，何诺生的语气突然变了。“明明，我才帮你想到一个好方法……”

    “什么？”我突然有种奇怪的预感……

    “我帮你想到办法了哦。”电话里，何诺生的声音充满男生特有的磁性，若是普通女生听到了都有可能被勾住心神……“不用交一百万，就能实现你大学梦想的办法。”

    “你……”我呆呆地拿着手机，惊讶得张大嘴巴说不出话――

    何诺生真的有办法帮我实现梦想……

    实现梦想！

    实现梦想！！！

    然后，第二天。

    何家米线馆的两位老板表情复杂地看着自家女儿，跟见了恐龙蛋一样完全说不出话。

    “爸爸，妈妈！”我被他们盯得心里发毛，什么啊，这是人家第一次剪短发穿男装耶！这种反应真的好吗？起码要让我转一个圈然后说我比何诺生还帅什么的吧……（自恋-_-||）

    “老婆大人，我是不是没睡醒？”爸爸居然目不转睛地盯着我，然后摸着妈妈的手，一副“我受到了惊吓”的表情。

    “老公，我受到了惊吓。”妈妈居然眼珠子不转地握紧爸爸的手，然后嘴巴张大成o型，跟见了鬼一样看着我……

    我把手握紧成拳头，忍无可忍地大吼：“你们够啦！！！”

    “娜娜，你这是要做什么？”爸爸妈妈都惊恐地抱在一起看着我……

    “我不管！”已经收拾好东西的我完全抛弃了酝酿了一晚上的离别话语，气鼓鼓地宣布我的想法：“你们都给我听好了！从现在开始，我就是何诺生，我要到比格利亚大学实现我的梦想！！！”

    “哈？？！”何家米线馆的两位老板都目瞪口呆地看着我……

    “哼！”我一鼓作气地拉着行李箱出了门，头也不回地迈着大步往外走。

    哼，本来想好好跟你们道个别。居然这样对我！

    另一边――

    何爸再次放下电话，松了口气。

    “怎么样？”

    “诺生说他有办法，我们就先相信他吧。”何爸抱着何妈的肩膀轻声安慰她。

    “嗯。”何妈这才放下心里的大石头。

    只要不找那个人，一切都好……

    地铁站。

    我看着周围大包小包的帅哥美女，再看看自己――

    一个简单的背包，一只比小孩子还大的行李箱，还是何诺生以前留下的，都有点旧了。

    别人还聚在一起有说有笑的，男的打扮得又高又帅像个男神，女的打扮得    性感    甜美整个女神……

    “唉……”我低着头叹了口气。果然是人比人，比死人……

    很快，地铁就来了。那些个帅哥美女看着车窗里面挤成人肉罐头的车厢都咽了下口水，然后小心翼翼地往里面钻。我看到这阵势也是醉了，敢情你们刚刚还聊啥苹果iPad笔记本的，现在这么挤还跟我们这些贫穷老百姓抢位置？！赶紧地拿着行李去打的啊！！！

    然而谁也没听到我内心的呼喊――大家上来才埋怨了一阵，又开始聊苹果iPad笔记本的了……

    好吧，我扶额～

    到了下一个站，貌似一些人下去了。我才要松口气，没想到一下子整个车厢的女生都开始了小声讨论，论着论着就变得大声了。我不解地看看周围，好像没什么啊？

    “你看，那个男生好帅啊……”

    “他好像在对着我笑，好害羞～”

    “好想问问他联系方式啊……”

    旁边一堆女生的傻笑声和花痴讨论引起了我的注意。谁啊？有那么夸张吗？我忍不住顺着她们的目光看过去，然后一下子呆在原地――

    那个男生穿着简单的天蓝色衬衫，一条白色休闲裤显得腿超长的。而且他好高，周围那些已经算有点帅的男生才刚到他的耳朵，看着快有180+了；那个男生戴着一只看着很高贵的手表，黑色马丁靴干净得发亮，背着深褐色的单肩包，看上去很有气质。他一直保持着微笑的表情，看着就像一位优雅又快乐的王子……

    我的天啊。顶级男神一枚！！！

    没错，他比刚才那些装逼的大头鬼们帅多了，现在整个车厢的女生们全都为他倾倒！

    嘿嘿，真可惜我现在是个男生……

    不然我也想好好发一把花痴，跟着那些姐妹们一起问他要联系方式啊……

    不过，男神就是男神。他很有礼貌地拒绝了那些女生的请求，只是继续戴着耳机陶醉在音乐里。

    我好奇地望着那枚薄荷绿耳机――那么好听吗？会是什么歌呢？

    他似乎注意到我的目光，突然转过头看着我，然后对我温柔地微笑。一瞬间，我整个人都被他电得大脑死机，无法运转――

    他的眼睛很漂亮，很少有男生像他一样画了眼线还能带出纯真干净的感觉；他的笑容好像会说话，我总感觉他在对我打招呼，好像我们很久以前就认识了……

    “……等我长大了，我就去找你啊！”

    “……长大了你就娶我好不好？”

    “傻娜娜……”

    一个男孩和一个女孩的对话突然间冲进我的脑海深处，然后我的头炸裂一样疼！！！

    我忍不住抱着头猛地蹲下身，旁边的行李箱砰地一声倒在了地上……

    那个女孩……那个男孩……到底是谁啊？！！

    “啊啊，好痛……”

    我痛苦地咬着下唇，脑海里满满的都是两个孩子一起打闹、一起聊天、一起谈心的画面……

    到底是谁，告诉我！

    周围人们的讨论声和惊呼都让我的心更加烦躁不安，刺眼的白光把我的最后一丝知觉带走了――

    闭上双眼的前一刻，我清楚地看到。

    一个满是柠檬香气的温软的怀抱接住了我。那人的声音听上去亲切又熟悉：“别害怕，我在。”

    是……是你吗？脑海里那个陌生又亲切的男孩？

    我好想出声问那个人，可是眼皮再也抬不起来……
